1.登录京东云控制台，选择“人工智能”->“认知服务”

![cms-80d99de9-ddf6-4b28-9b0d-140f028b11b120170524170411.png](https://img1.jcloudcs.com/cms/84cd2725-85e9-438e-8f95-fa6d5aa105d920170726183435.png)

2.打开控制台认知服务页面，进入应用列表页面。

![](https://img1.jcloudcs.com/cms/934007e5-85e3-4999-8994-d0885052c4d620170524170416.png)

3.创建应用：点击“创建应用”，填写英文和数字组合的应用名称，选择该应用需要使用的认知服务的接口类型，如下图，选择“智能鉴黄”接口（请根据业务需要选择需要的接口类型）：

![](https://img1.jcloudcs.com/cms/ea08cb68-7063-4493-8d40-84b8496ba62320170524170424.png)

注意：公测结束后，认知服务将为客户提供并发能力保障的服务，创建页面将根据用户选择的接口类型、计费类型生成服务订购订单。

4.应用创建完成后，应用列表中生成客户创建的认知服务应用。

![](https://img1.jcloudcs.com/cms/b25af8b7-c374-4c3b-a3e7-a7242bc90ef220170524170432.png)

5.接口调测：点击应用的“接口调试”按钮，可通过web-ui进行接口验证测试。

![](https://img1.jcloudcs.com/cms/1aeec215-e5b4-4643-94e2-b64654d2dd0c20170524170438.png)

如上所示，在用户创建的应用下，选择远程URL或者本地文件方式提交一张图片，点击调试按钮后，右侧栏中将实时显示智能鉴黄接口的鉴定结果。结果以JSON形式显示，鉴定结果参数如下：

{

"detectResult": [

{

"file": "2016.jpg",

"normalScore": 0.9932458996772766,

"hotScore": 0.004903233610093594,

"pornScore": 0.0018509498331695795,

"errCode": 0,

"errMsg": "ok"

}

],

"httpCode": 200,

"msg": "请求成功",

"timestamp": 1496199953648

}

errCode为0表示能够成功识别和鉴定；

normalScore参数表示图片是正常图片的概率；

hotScore参数表示图片是性感图片的概率；

pornScore 参数表示图片是色情图片的概率；

通过接口返回结果的解析，调用方可以获知指定图片的鉴定结果，通过概率大小识别是否为色情图片。
