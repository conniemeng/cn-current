# **数据恢复**

数据恢复功能可以最大程度地减少用户对数据库误操作引起的损失。目前云数据库 MongoDB 提供以备份数据的方式来恢复数据。

注意：数据恢复为覆盖式恢复，请谨慎操作，建议您在数据恢复前，手动创建一个备份文件。

**操作步骤：**

1、登录云数据库 MongoDB 控制台。

2、选择目标实例，进入实例详情页。

3、点击“备份恢复”，找到需要恢复的备份文件。

4、在备份文件的操作项中，点击“数据恢复”，在弹窗中再次确认。
