# **地域选择**

云数据库 MongoDB 默认提供三副本高可用架构（1主1从1隐藏），其中主从副本部署在同一个可用区，隐藏副本支持选择部署在其他可用区。您可以根据业务情况选择不同的部署方式。

**不同部署方式的特点：**

单可用区部署，可以承受机架级别的故障；

多可用区部署，可以承受机房级别的故障，但因为多个可用区之间存在一定的网络延迟，对于单个更新的响应时间会比单可用区高可用实例长。
