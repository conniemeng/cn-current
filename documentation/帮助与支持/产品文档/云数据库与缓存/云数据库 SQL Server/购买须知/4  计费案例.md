3.用户于“2017-08-15 15:20:30”删除资源

4.用户总共使用时长为5天1小时4分6秒，其中1小时4分6秒的时长折算为60/*(60+4)+6=3846秒，该时长费用为108/(3600/*24) /* 3846=4.81元

**5.**因此用户合计扣费为108/*5+4.81=544.81元
