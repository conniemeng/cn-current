**标签管理**

可以在【管理】--> 【标签管理】中查看所有资源的标签情况

详细的帮助，可以查看[标签管理](https://www.jdcloud.com/help/detail/3124/isCatalog/1 "标签管理")的相关文档

**标签列表**

可以标签列表中查看，搜索所有的标签及其绑定的资源
