*说明：*

* 
*KEYS命令，只能在VPC网络下使用，属于危险的命令，可能造成性能问题，请确保在key很少的情况下使用。如果需要从一个大数据集中查找特定的 key ，推荐使用 Redis 的集合结构(set)*
* 
*INFO命令，在集群版中使用受限：不支持 info cpu/replication/persistence*
* 
*CONFIG GET命令，在集群版中返回参数暂时未进行叠加*
